Step 2:Create a Key Pair in AWS (for SSH Access) and Set Proper File Permissions on the Key

aws ec2 create-key-pair --key-name cloudnova-dev-key --query "KeyMaterial" --output text > cloudnova-dev-key.pem

To create key-pair cloudnova-dev-key and saves the private key to cloudnova-dev-key.pem

icacls cloudnova-dev-key.pem /inheritance:r
icacls cloudnova-dev-key.pem /grant:r "%USERNAME%:R"

This command:Removes inheritance and grants read permission to the current user on the key file.

Terraform Installation Guide (Windows)

Step 1: Download Terraform

Download and install Terraform on your Windows system for AWS infrastructure management.

  • Go to Terraform Downloads Page
  • Download the Windows AMD64 zip file
  • Extract the zip file to a folder (e.g., C:\terraform)
  • Add Terraform to your PATH environment variable

Step 2: Environment Setup

Configure the PATH variable and verify the Terraform installation:

  • Press Windows + R, type sysdm.cpl, and press Enter
  • Click on Environment Variables
  • Under System Variables, find and select Path
  • Click Edit → New
  • Add the path where Terraform was extracted (e.g., C:\terraform)
  • Click OK on all dialogs to save changes

Step 3: Verify Installation

Confirm Terraform is properly installed and accessible:

  • Open Command Prompt or PowerShell
  • Run the command: terraform --version
  • Check that the version information is displayed
  • This confirms that the PATH variable is configured correctly

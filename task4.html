Step 4:Connection Method- AWS Console:

Click "Connect" button → Choose "EC2 Instance Connect" tab & Click "Connect" to open browser terminal

Step 5:Configure the EBS Volume:

Once connected, run these commands in order:

List available disks: lsblk

Format the volume (first time only): sudo mkfs -t xfs /dev/sdf

Create mount directory: sudo mkdir /mnt/additional-storage

Mount the volume: sudo mount /dev/sdf /mnt/additional-storage

Verify mount: df -h

10GB EBS volume is now attached and ready to use at /mnt/additional-storage/
